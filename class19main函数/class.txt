main函数起始也是一个函数，是主程序运行时默认调用的参数
main函数有默认传递的参数
int mian (int argc,char**argv)
argv为字符串类型，argc是argv的长度
argv默认有一个值为编译文件所处的位置

在powershell中进入编程文件文件夹
输入c++ 文件名
即可编译文件，编译结果名字默认为a
输入./编译文件名即可运行文件
输入./编译文件名 要输入的字符串
即可以空格为分隔符输入进argv中
所以
int main(int a,char**b){
    for(int i=1;i<a;i++){
        cout << b[i]<<endl;
    }
}
即可输出输入的字符串
同理，可以用这个功能直接在命令行中输入
